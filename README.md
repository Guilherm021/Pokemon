# 🎮 Jogo Pokémon em Java

Projeto acadêmico desenvolvido durante o semestre atual da faculdade com foco no aprendizado de Java utilizando IntelliJ IDEA.

## 📚 Objetivo

Este projeto foi criado para praticar e consolidar conhecimentos em:

- Programação Orientada a Objetos (POO)
- Lógica de programação
- Estruturação de projetos em Java
- Versionamento com Git e GitHub
- Organização de código

## 🚀 Funcionalidades atuais

- Escolha de Pokémon
- Sistema de batalha por turnos
- Ataque com chance de crítico
- Sistema de tipos (vantagem e desvantagem)
- Uso de poções
- Inimigo aleatório

## 🛠 Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git
- GitHub

## 📈 Próximas melhorias

- Sistema de XP e evolução
- Implementação de níveis
- Interface gráfica
- Sistema de múltiplas batalhas
- Organização em pacotes
