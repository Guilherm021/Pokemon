# 🎮 Jogo Pokémon em Java

Projeto acadêmico desenvolvido durante o curso de **Análise e Desenvolvimento de Sistemas**, com foco no aprendizado de **Java**, **Programação Orientada a Objetos (POO)**, lógica de programação e evolução contínua de software utilizando **Git/GitHub**.

---

## 📚 Objetivo do Projeto

Este projeto foi criado para praticar e consolidar conhecimentos em:

* Java
* Programação Orientada a Objetos (POO)
* Estruturas de decisão e repetição
* Organização de código
* Lógica de programação
* Versionamento com Git e GitHub
* Desenvolvimento incremental de software
* Interface gráfica com Java Swing

---

## 🚀 Funcionalidades Implementadas

## ⚔️ Sistema de Batalha

* Batalha por turnos
* Escolha de ataques
* Ataques críticos
* Sistema de vantagem de tipos:

  * 🔥 Fogo > 🌿 Planta
  * 🌿 Planta > 💧 Água
  * 💧 Água > 🔥 Fogo

---

## 🧪 Sistema de Itens

* Uso de poções durante a batalha
* Recuperação de vida
* Loja com PokéCoins
* Compra de melhorias

---

## 📈 Sistema de Progressão

* Ganho de XP por vitória
* Sistema de níveis
* Evolução automática dos Pokémon
* Aprendizado de novos ataques
* Aumento de atributos ao subir de nível

---

## 👑 Desafio Final

* Progressão até batalha final
* Chefão especial: **Mewtwo**

---

## 🏆 Sistema de Ranking

* Recorde de vitórias
* Maior nível alcançado

---

# 🖥️ Versões do Projeto

## 💻 Versão Console

Arquivo principal:

```text
JogoPokemon.java
```

Versão jogável diretamente no terminal/console com menus interativos.

---

## 🪟 Versão Interface Gráfica

Arquivo principal:

```text
JogoPokemonGUI.java
```

Versão com interface gráfica utilizando **Java Swing**, contendo:

* Janela interativa
* Botões de ataque
* Barra de vida visual
* Status do jogador e inimigo
* Log de batalha em tempo real
